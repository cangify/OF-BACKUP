# OF-BACKUP / Onlyfans Backup

OF-BACKUP 是一款用于个人授权内容备份与整理的桌面软件。它可以帮助用户在本机整理当前账号可访问的作者、帖子、图片和视频，并批量加入下载队列保存到本地。

> 重要说明：本工具只用于备份当前账号可见且可直接访问的内容，不绕过平台权限、付费限制或 DRM 保护。请仅在你拥有合法访问权限和备份权限的情况下使用。

## 主要功能

- **账号登录检测**：显示当前登录账号状态，支持清除本机登录状态。
- **作者列表自动加载**：打开软件后会恢复上次作者列表，并在已登录状态下自动后台刷新。
- **作者筛选与搜索**：支持作者搜索、多选、有效订阅/已过期订阅筛选。
- **媒体扫描**：选择作者或输入 `@用户名` / 链接，扫描当前账号可访问的备份内容。
- **批量下载队列**：可把选中的图片和视频加入下载队列，自动跳过已存在文件。
- **下载中心**：查看总进度、实时速度、任务状态、失败重试、打开保存目录。
- **任务状态筛选**：支持按全部、正在下载、等待中、已暂停、成功、已存在/跳过、失败筛选任务。
- **下载设置**：可配置保存位置、并发下载数量、扫描时长和视频清晰度偏好。
- **中英文切换**：界面支持中文 / English 切换。

## 使用提示

1. 打开软件并登录账号。
2. 等待作者列表自动恢复/刷新，或手动点击“刷新作者列表”。
3. 选择作者并扫描媒体。
4. 选择需要保存的内容，点击“加入下载队列”。
5. 软件会自动跳转到“下载中心”，可查看进度、速度和任务状态。

## 合规声明

OF-BACKUP 不提供绕过权限、破解 DRM、抓取不可访问内容的能力。软件仅处理用户当前账号可见且可直接访问的内容。使用者需要自行遵守相关平台条款、当地法律法规及内容权利人的要求。
